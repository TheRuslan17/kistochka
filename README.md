# 🎨 Кисточка

![Версия](https://img.shields.io/badge/версия-1.3-green)
![Лицензия](https://img.shields.io/badge/лицензия-Freeware-blue)
![Windows](https://img.shields.io/badge/platform-Windows-lightgrey)

**Кисточка** — простая и удобная программа для рисования на Windows. Создавайте рисунки с помощью кисти, фигур и заливки. Поддержка отмены действий (Ctrl+Z) и сохранения в популярные форматы.

## ✨ Возможности

- 🖌️ **Кисть** — рисование от руки с регулируемым размером
- 🧼 **Ластик** — стирание линий
- 🔷 **Фигуры** — прямоугольник, эллипс, треугольник, линия, стрелка
- 🎄 **Елочка** и ⭐ **Звезда** — праздничные фигуры
- 🪣 **Заливка** — закрашивание замкнутых областей
- ↩️ **Отмена действий** (Ctrl+Z) и возврат (Ctrl+Y)
- 💾 **Сохранение** в PNG, JPG, BMP
- 📏 **Регулировка размера кисти**
- 🎨 **Выбор цвета** через стандартный диалог Windows

## 🚀 Установка

1. Скачайте последнюю версию: **[Скачать Кисточка 1.3](https://disk.yandex.ru/d/UJ3b6suKOoxoLQ)**
2. Распакуйте архив (если скачали ZIP)
3. Запустите `Кисточка.exe`

*Программа не требует установки — просто запустите и рисуйте!*

## 🎮 Как пользоваться

| Кнопка | Действие |
|--------|----------|
| 🎨 Цвет | Выбор цвета кисти |
| 🖌️ Кисть | Рисование |
| 🧼 Ластик | Стирание |
| 🔷 Фигуры | Выбор фигуры из меню |
| 🪣 Заливка | Заливка области |
| 🧹 Очистить | Очистка холста |
| 📄 Новый | Создать новый файл |
| ↩️ Отмена | Отменить последнее действие (Ctrl+Z) |
| ↪️ Вернуть | Вернуть отменённое (Ctrl+Y) |
| 💾 Сохранить | Сохранить рисунок |

## ⌨️ Горячие клавиши

- `Ctrl+Z` — отмена действия
- `Ctrl+Y` — возврат действия

## 📋 Системные требования

- Windows 7, 8, 10, 11
- .NET Framework 4.6.1 или выше (обычно уже установлен в Windows)
